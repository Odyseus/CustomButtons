This is a simple button that creates a panel with shortcuts (menuitems) to websites. I created this button because I was looking for a way to replace an extension that I used as my homepage (Super Start). I called it Speed Dial due to the fact that I've added to it keyboard shortcuts to access the sites.

Features

  • Ability to set up to 62 hotkeys (0-9, a-z, A-Z). Hotkeys can be accessed/triggered only when the panel is open.
  • Possibility to open the panel with a hotkey. Configurable when editing the button ("Button settings" tab).
  • The appearance of the panel and menuitems can be customized with CSS through the Help tab of this button.
  • Although not recommended or practical, an unlimited number of items/sites can be added.

Options (Acceced by middle-clicking Speed Dial button.)

  • Open on mouseover: Open Panel on mouseover.
  • Open/Close delay: Open/Close panel delay in miliseconds. Only used when "Open on mouseover" is set to true/checked.
  • Show dropmarker: Shows/Hides this button dropmarker.
  • Panel options:
    • Hide panel on hotkey pressed: This option, if set to false/unchecked, allows to press more than one hotkey to open more than one site before the panel is closed. Only used when "Use items hotkeys" is set to true/checked.
    • Number of columns: Divide menu in columns. Possible values 1 to 5.
    • Open tabs in background: If set to true/checked, tabs open in background. If set to false/unchecked, tabs open in foreground.
    • Panel flip attribute: If set to true/checked, sets the flip attribute of the panel to the value "both". If set to false/unchecked, sets the flip attribute of the panel to the value "slide". More info about the flip attribute in panels.
    • Use images on items: Shows/Hides images of menuitems inside the panel.
    • Use items hotkeys: Enables/Disables the ability to open a menuitem (webpage) with a hotkey.
  • Tab override options:
    • Drastic tab overwrite: If set to true/checked and the current tab is any "about:" page, the tab will be overwrited. If unchecked, only tabs specified in "OverrideURLs" and/or "OverrideHosts" will be overwrited.
    • Hosts to override: List of servers/host names to override (comma separated and no spaces). If the currently loaded page contains one of these servers/host names in its URL, the page will be opened on the current tab.
    • URLs to override: List of URLs to override (comma separated and no spaces). If the currently loaded page is one of these URLs, the page will be opened on the current tab.
  • Updates checker options:
    • Auto-check for updates: If set to true/checked, Speed Dial will auto-check for updates every x amount of days.
    • Check every...: Sets the amount of days for Speed Dial to auto-check for updates.
    • Check delay...: Sets the amount of seconds after application startup for Speed Dial to start looking for updates.
    • Check now: Check for updates right away.

Compatibility

This button is intended to work with all Firefox-based browsers and SeaMonkey. But it has been tested only in the following environments.

Arch Linux (XFCE and KDE) - Firefox 34.x
Windows 7 x64 - Palemoon x64 25.x, Cyberfox x64 34.x and latest versions of Firefox ESR/Stable/Beta

How to add sites

Sites are added by editing the Initialization tab of this button.

Examples

sites.push(['Site Name', 'Site URL', 'Site Favicon/Image', 'Hotkey']);
sites.push(['Google Search', 'https://www.google.com', 'https://www.google.com/favicon.ico', 'g']);

The site Favicon/Image can be:

  • An image URL hosted on internet (https://translate.google.com/favicon.ico).
  • A local image (file:///C:/image.png).
  • A "Firefox image" (chrome://branding/content/about-logo.png).
  • A base64 encoded image (data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAACAAAAAgCAYAAABzenr0AAAFw0lEQVRYhc3XfUyUBQDH8d).

About hotkeys

  • Only 1 digit alphanumeric characters (0-9, a-z and A-Z ) without modifiers (NO Ctrl, NO Shift, NO Alt and NO Win keys).
  • Case sensitive. This means that "a" and "A" are two different hotkeys.
  • There can not be 2 identical hotkeys. Doh!
  • Hotkey will be reflected in item's label. More info about "accesskey" atribute.

About CSS customization

The CSS style of this button can be edited in the Help tab. It can be found inside two "markers" called <!--StyleSheet--> (NEVER edit this markers). Inside those "markers" you can add any CSS code to style this button or any part of the browser.
Defalut CSS style:

<!--StyleSheet-->
#_thisButtonId_-panel .panel-arrowcontent,
#_thisButtonId_-panel .panel-subviews { /* Style the panel itself */
	background: -moz-linear-gradient(0% 75% 90deg, rgb(207, 204, 201), rgb(224, 222, 221)) !important;
}
#_thisButtonId_-panel .panel-arrowbox { /* Remove panel arrow */
	display: none !important;
}
#_thisButtonId_ .panel-arrowcontainer { /* Remove panel animation */
	transform: none !important;
	transition: none !important;
	pointer-events: all !important;
}
menuitem._thisButtonId_-bookmark-items { /* Menuitems inside the panel */
	-moz-appearance: none !important;
	font-size: 12px !important;
	font-weight: bold !important;
	color: rgb(0, 0, 0) !important;
	border-radius: 3px !important;
	padding: 4px  !important;
	margin: 5px  !important;
	box-shadow: inset 0 0 0px 1px rgba(255, 255, 255, .5), 0 1px 2px 0px rgba(0, 0, 0, .6) !important;
	background: -moz-linear-gradient(0% 75% 90deg, rgba(207, 204, 201, .4), rgba(224, 222, 221, .4)) !important;
	border: none !important;
}
menuitem._thisButtonId_-bookmark-items .menu-iconic-icon { /* Menuitems image/icon */
	width: 32px !important;
	height: 32px !important;
	/*margin: 2px  !important;*/
}
menuitem._thisButtonId_-bookmark-items .menu-iconic-text { /* Menuitems label */
	padding-right: 5px  !important;
}
menuitem._thisButtonId_-bookmark-items:hover { /* Menuitems when hovered */
	box-shadow: 0 1px 2px 1px rgba(0, 0, 0, .6) !important;
	cursor: pointer !important;
	-moz-transition-duration: 0.2s !important;
}
menuitem._thisButtonId_-bookmark-items:active { /* Menuitems when activated (clicked) */
	box-shadow: inset 0px 1px 1px 0px rgba(0, 0, 0, .4), 0px 1px 0px 0px rgba(255, 255, 255, .4) !important;
	border: none !important;
	-moz-transition-duration: 0.2s !important;
}
<!--StyleSheet-->

Note: The text called _thisButtonId_ is another "marker" or "placeholder" (NEVER edit this markers). It will be replaced "on the fly" by the ID of this button, so the style of this button does not interfere with the styles of the rest of the browser.

Error report

If any errors are found, report it ASAP on this button homepage with as much details as possible.
  • Wich plattform and its version (Linux, Mac or Windows).
  • Wich application and its version (Any Firefox-based browser or SeaMonkey).
  • Error/Browser console output, if any.
  • If it's easier for you to report an error with images, and if the image is too large or there are more than one, publish the links to them, do not post them using the IMG tag in the message.
  • Be willing to follow troubleshooting instructions.
