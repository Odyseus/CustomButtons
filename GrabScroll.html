Install this button

GitHub

This button is based on a script created for userChrome.js extension. The author of the script is Alice0775 (Source). When enabled, GrabScrol will allow you to "scroll" the current page by dragging its content.

Button clicks

  • Left click: Enables/Disables Grabscroll.
  • Middle click: Open options menu.

Options

  • Button to start the GrabScroll: Possible values "L", "M" or "R" (Left, Middle or Right click).
  • GrabScroll start threshold: Stop GrabScroll if mouse button is pressed longer than this time (In milliseconds).
  • GrabScroll speed & direction: Drag direction and speed. With a negative value, the page content moves in the same direction as the cursor. The bigger the number, the faster the scroll speed is.
  • Scroll inertia: If enabled, the page wil continue to scroll for a short while.

  • Cancel GrabScroll on images: Do not start GrabScroll if cursor is over an image.
  • Cancel GrabScroll on links: Do not start GrabScroll if cursor is over a clickable link.

Compatibility

This button is intended to be cross-platform (GNU/Linux-Mac OS-Windows) and to work with all Firefox-based browsers and SeaMonkey. But it has been tested only in the following environments.

Arch Linux (XFCE and KDE) - Firefox 33.x
Windows 7 x64 - Palemoon x64 25.x.x, Cyberfox x64 33.x, SeaMonkey 2.30.x and latest versions of Firefox ESR/Stable/Beta
