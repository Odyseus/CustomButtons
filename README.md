### Requirements to build the HTML pages from the PUG files
THe **src** folder contains the **pug** files to build the **HTML** files.

- Node.js
    - Download the [source](https://nodejs.org/download/) and install.
    ```bash
    $ ./configure
    $ make
    $ [sudo] make install
    ```

- Pug
    ```bash
    $ npm install pug-cli -g
    ```
