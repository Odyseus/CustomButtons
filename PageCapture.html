Install this button

GitHub

This button is based on code found in Mozilla German Forums (Source). It allows to capture an image of the current page content.

Options

Options for this button can be accessed by editing the Initialization tab code.

  • ContextMenuForImages: If set to true, it will add a context menu for images that, when clicked, will copy to the clipboard the selected image in base64 format.
  • IsMenu:
    • True: This button will display a menu with four possible actions. See below.
    • False: This button will perform one of four possible actions. See below.

Possible actions and button clicks

All actions performed will open the captured image in a new tab.

  • Capture whole page (Left click): It will capure the whole content of the current loaded page.
  • Capture current view (Ctrl + Left click): It will capure only the visible content of the current loaded page.
  • Capture selected region (Middle click): It will allow you to select a region of the current loaded page to be captured.
  • Capture selected node (Ctrl + Right click): It will allow you to select an HTML node of the current loaded page to be captured.

Compatibility

This button is intended to be cross-platform (GNU/Linux-Mac OS-Windows) and to work with all Firefox-based browsers and SeaMonkey. But it has been tested only in the following environments.

Arch Linux (XFCE and KDE) - Firefox 33.x
Windows 7 x64 - Palemoon x64 25.x.x, Cyberfox x64 33.x, SeaMonkey 2.30.x and latest versions of Firefox ESR/Stable/Beta

Error report

If any errors are found, report it ASAP on this button homepage with as much details as possible.
  • Wich plattform and its version (Linux, Mac or Windows)
  • Wich application and its version (Any Firefox-based browser or SeaMonkey)
  • Error/Browser console output, if any.
  • If it's easier for you to report an error with images, and if the image is too large or there are more than one, publish the links to them, do not post them using the IMG tag in the message.
  • Be willing to follow troubleshooting instructions.
