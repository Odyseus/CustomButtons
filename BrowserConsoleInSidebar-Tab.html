This button is not my creation. I found this button on the Mozilla Russian forum a couple of months ago. The original author is Dumby and I found it posted by bunda1 here.

Install Browser Console in Tab/Sidebar (original version by Dumby)

Shortcuts

  • Left Click = Open Browser Console in a new tab (or switch to it if already open).
  • Middle Click = CustomButtons context menu.
  • Right Click = Open Browser Console in Sidebar.

The version tweaked by me (Odyseus), can be installed from the "Install this button" button to the right of this page.

Features added by me

  • Button clicks rearranged.
  • The console remains open after browser restart (Only if All-In-One Sidebar addon is installed).
  • The console is colorized through a style sheet.
  • Changed the default icons for the console messages.
  • Maximum/Minimum width restriction of sidebar removed.
  • The color, icons and width restriction can be customized/removed by editing the help tab of this button.
