GitHub

This button is based on a script created for userChrome.js extension (Source). It allows to override the browser's current User Agent by selecting a new one from a predefined list. The list of available User Agents can be edited by editing the Help tab of this button.

The list of user agents is in JSON format, wich is very strict. A single misplaced comma or double quotes and the button will simply break. The "rules" are very simple:

  • Each block (curly braces) is separated by a comma (,), except the last block.
  • Each line inside each block ends with a comma (,), except blocks with just one line (like the separators) or the last line in case of blocks with more than one line.

Example


[{
		"label" : "Name of User Agent",				// COMMA
		"agent" : "User Agent itself"				// NO COMMA
	}, {
		"label" : "separator"					// NO COMMA
	}, {
		"label" : "Name of User Agent",				//  COMMA
		"accesskey" : "A single digit key (a-z, A-Z or 0-9)",	// COMMA
		"agent" : "User Agent itself"				// NO COMMA
	}								// NO COMMA
]

Compatibility

This button is intended to be cross-platform (GNU/Linux-Mac OS-Windows) and to work with all Firefox-based browsers and SeaMonkey. But it has been tested only in the following environments.

Arch Linux (XFCE and KDE) - Firefox 33.x
Windows 7 x64 - Palemoon x64 25.x.x, Cyberfox x64 33.x, SeaMonkey 2.30.x and latest versions of Firefox ESR/Stable/Beta

Error report

If any errors are found, report it ASAP on this button homepage with as much details as possible.
  • Wich plattform and its version (Linux, Mac or Windows)
  • Wich application and its version (Any Firefox-based browser or SeaMonkey)
  • Error/Browser console output, if any.
  • If it's easier for you to report an error with images, and if the image is too large or there are more than one, publish the links to them, do not post them using the IMG tag in the message.
  • Be willing to follow troubleshooting instructions.
